# Airfocus - Complete Feature Inventory

## Core Philosophy: Modularity

Airfocus is built on the principle that every team works differently. Instead of forcing teams into rigid structures, the platform allows complete customization of:
- Workspaces
- Views
- Fields
- Workflows
- Roadmap formats
