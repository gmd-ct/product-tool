# Airfocus - Users, Personas & Use Cases

## Target Users

### Primary Personas

#### 1. Product Managers
**Role**: Day-to-day roadmap and prioritization management

**Daily Activities**:
- Prioritize features using scoring frameworks
- Build and update roadmaps
- Collect and process feedback
- Align work with OKRs
- Communicate plans to stakeholders

**Why They Choose Airfocus**:
- Priority Poker for team consensus
- Flexible scoring frameworks
- Customizable roadmap views
- Strong Jira integration

#### 2. Product Directors/Leaders
**Role**: Strategic oversight and portfolio management

**Daily Activities**:
- Review multi-product portfolios
- Ensure alignment with company objectives
- Report to executives
- Manage product team resources

**Why They Choose Airfocus**:
- Portfolio rollup views
- OKR alignment features
- Customizable hierarchies
- Multi-workspace management

#### 3. Product Operations Managers
**Role**: Process and tool standardization

**Daily Activities**:
- Configure and maintain airfocus
- Create templates and standards
- Train product managers
- Manage integrations

**Why They Choose Airfocus**:
- Deep customization capabilities
- Template creation
- Workspace governance (Scale)
- Integration management

#### 4. Project Managers
**Role**: Execution and delivery coordination

**Daily Activities**:
- Track project progress
- Manage capacity
- Coordinate cross-team work
- Report on delivery

**Why They Choose Airfocus**:
- Capacity planning features
- Jira/Azure DevOps integration
- Timeline views
- Progress tracking

## Use Case Examples

### Use Case 1: Modular Product Portfolio Management

**Scenario**: Large organization managing 30+ products needs unified system

**Customer**: Papirfly

**Workflow**:
1. Create hierarchical workspace structure
2. Customize fields for each product type
3. Build tailored roadmap views
4. Configure portfolio rollups
5. Link OKRs across organization
6. Track progress at all levels

**Outcome**: "Built unified setup for 32 products and 300+ services in 12 months, providing constant clarity around product vision & strategy."

### Use Case 2: Team Prioritization with Priority Poker

**Scenario**: Product team needs consensus on quarterly priorities

**Workflow**:
1. Add all candidate features to backlog
2. Configure scoring criteria (impact, effort, etc.)
3. Invite team to Priority Poker session
4. Team members vote on each criterion
5. System calculates consensus scores
6. Review and finalize priorities
7. Add top items to roadmap

**Value**: "The priority scoring system has been incredibly impactful, helping identify high-value tasks aligned with strategic goals."

### Use Case 3: Feedback-to-Roadmap Workflow

**Scenario**: PM needs to connect customer feedback to product decisions

**Workflow**:
1. Integrate Intercom/Zendesk for support feedback
2. Connect Salesforce for sales insights
3. Collect direct feedback via portal
4. Centralize all feedback in airfocus
5. Link feedback to roadmap items
6. Use volume/value to inform prioritization
7. Close the loop with customers

**Value**: "Airfocus has been helpful in collecting user feedback in one place and making it easy to connect that feedback to roadmap initiatives."

### Use Case 4: Jira-Connected Product Management

**Scenario**: Align product strategy with engineering execution

**Workflow**:
1. Connect airfocus to Jira Cloud
2. Configure custom field mapping
3. Create roadmap in airfocus
4. Push epics/features to Jira
5. Track Jira status from airfocus
6. Link OKRs to user stories
7. Maintain bi-directional sync

**Value**: "The integration with JIRA is very very good - it allows for custom mapping and an easy to use interface."

### Use Case 5: EU-Compliant Product Management

**Scenario**: European company needs GDPR-compliant platform

**Customer**: Oodrive

**Workflow**:
1. Select EU data hosting option
2. Configure GDPR-compliant settings
3. Set up SSO/SAML (Enterprise)
4. Leverage ISO 27001 certification
5. Document compliance for audits

**Value**: "Praised EU-based hosting and flexibility for multi-team streamlining."
