# Airfocus - Pricing Analysis

## Pricing Overview

Airfocus uses per-editor pricing with three main tiers. The platform emphasizes flexibility and offers special pricing for startups, nonprofits, and educational organizations.

## Current Pricing Tiers (2025)

| Plan | Price | Billing | Best For |
|------|-------|---------|----------|
| **Core** | $59/editor/month | Annual | Small teams, essential features |
| **Scale** | Custom | Annual | Growing teams, API access |
| **Enterprise** | Custom | Annual | Large organizations, advanced security |

## Detailed Plan Comparison

### Core Plan ($59/editor/month)
**Best For**: Small product teams needing essential roadmapping and prioritization

**Included**:
- Roadmapping
- Prioritization (Priority Poker, scoring)
- Feedback collection
- Basic integrations
- Standard support
- Unlimited workspaces
- Unlimited roadmaps

### Scale Plan (Custom Pricing)
**Best For**: Growing teams needing advanced features and integrations

**Included**:
- Everything in Core
- API access
- Real-time webhooks
- Advanced integrations (Jira, Azure DevOps, etc.)
- Workspace governance
- Advanced reporting
- Priority support

**Integrations Included**:
- Jira Cloud
- Azure DevOps Cloud
- Linear
- Trello
- Asana
- Shortcut
- MS Planner
- GitHub
- Zapier
- Intercom
- Slack
- MS Teams
- Zendesk

### Enterprise Plan (Custom Pricing)
**Best For**: Large organizations with security and compliance requirements

**Included**:
- Everything in Scale
- Enhanced workflows
- Advanced governance
- SSO/SAML
- Advanced security controls
- Dedicated support
- Custom terms
- SLA guarantees

## Billing Options

### Flexibility
- **Annual Plans**: Standard option, discounted
- **Monthly Plans**: Available, can terminate at end of billing month
- **Plan Changes**: Can upgrade/change anytime (pro-rated)
- **Annual Termination**: At end of contract year

### Trial
- **Duration**: 14 days
- **Credit Card**: Not required
- **Features**: Full access during trial

### Value Analysis

**Airfocus vs. Productboard**:
- Airfocus Core ($59) vs. Productboard Pro ($59) = Similar
- Airfocus emphasizes modularity
- Productboard emphasizes feedback

**Airfocus vs. Aha!**:
- Similar entry pricing ($59 vs $59)
- Aha! more comprehensive but steeper learning curve
- Airfocus more flexible/modular

**Airfocus vs. ProductPlan**:
- Airfocus more expensive at entry
- Airfocus more features
- ProductPlan simpler, cheaper
