# Airfocus - Integration Ecosystem

## Integration Overview

Airfocus offers 15+ native integrations plus API access (Scale plan). The platform emphasizes development tool connectivity and workflow automation.

## Integration Strengths

### Jira Integration Excellence
- Two-way sync
- Custom field mapping
- Easy-to-use interface
- Strategy-to-development connection

### Workflow Automation
- Zapier connectivity
- Real-time webhooks
- Custom automation building

## Integration Limitations

### Plan Restrictions
- Most integrations require Scale plan
- Core plan has limited integration options
- API access only on Scale+

### Missing Integrations
- No native Figma integration
- Limited analytics tool connections
- No native Amplitude/Mixpanel

## Integration Best Practices

### Recommended Setup Priority
1. **Jira/Azure DevOps** - Development alignment (Scale)
2. **Slack/Teams** - Team communication
3. **Intercom/Zendesk** - Feedback capture
4. **Zapier** - Fill integration gaps

### Power User Tips
- Use custom field mapping in Jira
- Set up Slack channels per workspace
- Leverage webhooks for custom automation
- Connect feedback sources early
