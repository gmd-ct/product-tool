- **Unlimited Roadmaps**: Create and manage without restrictions

- **Color-Coded Strategies**: Visual differentiation
- **Custom Legends**: Define your own color meanings
- **Custom Tags**: Organize with tagging system
- **Containers**: Group related items
- **Bars**: Individual initiative representation
- **Lanes**: Horizontal organization (projects/themes)
- **Milestones**: Key date markers
- **Dependencies**: Link related items
- **Strategic Dependencies View**: Visualize cross-team dependencies
- **Master Plan**: Merge multiple roadmaps for big picture view
- **Multiple Versions**: Create unlimited roadmap versions

- **Export Options**: PDF, image, spreadsheet formats

## Feature Gaps & Limitations

### Missing Features
- No native changelog functionality
- No automated notification emails to feedback submitters
- Limited advanced analytics
- No built-in backlog feature (separate from parking lot)
- No native capacity planning
- No time tracking
- Limited customization of fonts/text placement

### Known Limitations
- Some editing functionality described as "finicky"
- UI noted as needing modernization by some users
- Less advanced than specialized tools like Roadmunk for roadmapping
- Lacks deep feedback management features
